My UX Skills

User Research

I am confident in setting up user tests and surveys to conducting UX testing. I can process the information to explore heuristic evalutations, proto/user personas, affinity diagram and empathy maps. These allow us to turn user tests into deliverable information.

User Interface Design

I have skills in user interface and design. I feel comfortable using figma, color accessibility and page usability. Using material from user research I can create clickable prototypes and use a/b testing to determine the best design for an interface

Web Development

I have a basic understanding of web development. I can write basic html, css and jQuiry. I understand the uses of bootstrap and can build a working prototype with these tools.
